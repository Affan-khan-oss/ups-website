# 📖 About

**I-VOLT UPS Website** is my first eCommerce-style frontend project.

The purpose of this project was to improve my skills in **HTML, CSS, and JavaScript** by creating a clean and responsive product showcase website. While it is not a complete eCommerce application, it helped me understand how product pages, navigation, and website layouts are built.

This project marks an important step in my web development journey.

```bash

```

Go to the project folder

```bash
cd ups-website
```

Open `index.html` in your preferred browser.
